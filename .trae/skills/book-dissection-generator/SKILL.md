# 书本拆解生成器

## 功能介绍

这是一个专业的书本、章节、文章拆解助手，专门用于生成个人学习、职场提升、能力精进类的公众号文章。

### 核心功能

1. **内容输入** - 支持输入书本名称、粘贴章节内容、文章链接
2. **核心提取** - 提取金句、核心观点、案例
3. **观点总结** - 提炼核心思想
4. **个人感悟** - 加入"快乐IT啊"的经历和感悟
5. **公众号文章生成** - 完整的公众号文章（固定开头+个人经验）
6. **模板匹配** - 4种拆解模板
7. **去AI味优化** - 让文章更像真人写作
8. **联网搜索** - 搜索书本相关资料（可选）

### 增强功能

9. **文件保存** - 将生成的文章保存为markdown文件
10. **多格式导出** - 支持markdown、纯文本等格式
11. **版本管理** - 保存多个版本，支持对比和回退
12. **SEO优化建议** - 提供关键词、摘要等优化建议
13. **标题A/B测试** - 生成多个标题，分析点击率潜力
14. **字数统计** - 自动计算字数和预估阅读时长
15. **文章迭代** - 支持逐步修改和优化文章

### 特殊要求（重要！）

16. **固定开头** - 所有文章开头必须是：【大家好，我是快乐IT啊。】
17. **大量个人经验** - 文章中必须加入丰富的个人读书经历、感受、感悟等
18. **作者人设** - "快乐IT啊"是一个有10年IT经验的程序员，喜欢分享，说话幽默风趣，热爱读书和学习

## 使用方法

### 快速开始

1. 告诉AI你想拆解什么内容（书名/章节/文章）
2. 提供内容（粘贴内容、说书名、或提供链接）
3. 选择或让AI自动匹配合适的拆解模板
4. 按流程生成：输入 → 提取 → 总结 → 感悟 → 文章
5. 完成后自动保存为markdown文件

### 拆解模板

内置4种拆解模板：

- **核心观点型**：固定开头 → 金句摘录 → 观点解读 → 个人感悟
- **方法实践型**：固定开头 → 方法步骤 → 案例分析 → 行动计划
- **思维成长型**：固定开头 → 概念介绍 → 启发思考 → 应用建议
- **综合解读型**：固定开头 → 背景介绍 → 核心内容 → 总结感悟
