# 增强功能详解

## 功能8：文件保存

### 功能说明
文章生成完成后，自动将文章保存为markdown文件到本地。

### 文件命名规则
- 格式：`articles/YYYY-MM-DD-{书名缩写}-dissection.md`
- 示例：`articles/2026-03-20-atomic-habits-dissection.md`

### 保存内容
包含完整的文章内容，以及：
- 元数据（书名、作者、拆解日期、字数、阅读时长）
- 书籍基本信息
- 核心观点清单
- 配图建议
- SEO优化建议
- 版本信息

### 使用示例
```
> 用户：帮我拆解《原子习惯》这本书
> AI：...（生成拆解文章）...
> AI：文章已保存为：articles/2026-03-20-atomic-habits-dissection.md
```

## 功能9：多格式导出

### 支持格式
1. **Markdown** (.md) - 推荐，保留格式
2. **纯文本** (.txt) - 去除所有格式
3. **HTML** (.html) - 可直接预览

### 导出命令
- "导出为markdown"
- "给我一个纯文本版本"
- "生成HTML格式"

---

## 功能10：版本管理

### 版本命名
- v1.0 - 初稿
- v1.1 - 小修改
- v2.0 - 重大调整

### 文件存储
```
articles/
├── atomic-habits-dissection/
│   ├── v1.0.md
│   ├── v1.1.md
│   └── v2.0.md
└── latest.md  (指向最新版本)
```

### 版本对比
可以要求：
- "对比v1.0和v2.0的差异"
- "回退到v1.0版本"
- "基于v1.1继续修改"

## 功能11：SEO优化建议

### 优化内容

#### 1. 关键词建议
- 主关键词（1-2个）：书名、作者、核心主题
- 长尾关键词（3-5个）：书名+读后感、书名+拆解、如何阅读+书名
- 相关关键词：个人成长、习惯养成、思维提升

#### 2. 摘要优化
- 150-200字的摘要
- 包含书名和核心主题
- 吸引人点击

#### 3. 标题优化
- 包含书名
- 20-30字最佳
- 有吸引力

#### 4. 正文优化
- 关键词密度建议
- 内部链接建议
- 图片ALT文字建议

### 输出示例
```
【SEO优化建议】

📌 主关键词：原子习惯、詹姆斯·克利尔
📌 长尾关键词：原子习惯读后感、原子习惯拆解、如何阅读原子习惯
📌 相关关键词：个人成长、习惯养成、自我提升

📝 推荐摘要：
读完《原子习惯》这本书，我彻底改变了对习惯的看法！
这篇拆解文章，帮你快速掌握书中的核心观点，
让你也能养成好习惯！

💡 优化建议：
- 在第一段就出现"原子习惯"
- 正文中自然提及关键词3-5次
- 给图片添加ALT文字
```

## 功能12：标题A/B测试

### 生成数量
通常生成3-5个备选标题

### 评分维度
1. **吸引力**（0-10分）- 是否想点击
2. **清晰度**（0-10分）- 是否知道讲什么
3. **关键词**（0-10分）- 是否包含书名
4. **综合评分**（0-10分）

### 输出示例
```
【标题A/B测试】

1. 《读完<原子习惯>，我彻底改变了对习惯的看法！》
   🎯 吸引力：9分
   📖 清晰度：10分
   🔑 关键词：10分（包含原子习惯）
   ⭐ 综合：9.7分
   💡 推荐理由："彻底改变"和个人经历很有吸引力

2. 《<原子习惯>拆解：为什么我读了3遍还想读？》
   🎯 吸引力：10分
   📖 清晰度：9分
   🔑 关键词：10分
   ⭐ 综合：9.7分

3. 《<原子习惯>的3个核心观点，帮你养成好习惯》
   🎯 吸引力：7分
   📖 清晰度：10分
   🔑 关键词：10分
   ⭐ 综合：9.0分

🏆 推荐使用第1个或第2个标题！
```

## 功能13：字数统计&阅读时长

### 统计内容
- **总字数** - 包含标点和空格
- **汉字数** - 仅统计汉字
- **段落数**
- **小标题数**

### 阅读时长预估
- 普通阅读：300字/分钟
- 快速阅读：500字/分钟
- 精读：150字/分钟

### 公众号建议字数
- 最佳：1500-2500字
- 短篇：800-1500字
- 长篇：2500字以上

### 输出示例
```
【字数统计】

📊 总字数：2156字
📝 汉字数：1892字
📄 段落数：42段
🏷️ 小标题数：8个

⏱️ 阅读时长预估：
- 普通阅读：约7分钟
- 快速阅读：约4分钟
- 精读：约14分钟

💡 建议：当前字数2156字，在最佳范围内（1500-2500字），继续保持！
```

## 功能14：文章迭代

### 迭代方式

#### 1. 整体修改
- "把开头改得更吸引人"
- "让文章更口语化一些"
- "增加更多个人经历"

#### 2. 局部修改
- "把第二段改一下"
- "第3部分的观点讲得不够清楚，重新写"
- "结尾再加个总结"

#### 3. 风格调整
- "改成更严肃的风格"
- "再活泼一点"
- "用第一人称写"

### 迭代流程
```
当前版本
    ↓
用户提出修改需求
    ↓
AI理解需求并修改
    ↓
生成新版本（v1.1）
    ↓
自动保存
    ↓
询问用户是否满意
```

### 功能说明

```
>      ⭐ 豆瓣评分：8.9分

```

## 功能16：章节/段落拆解

### 功能说明
除了整本书拆解，还可以：
- 只拆解某一章节
- 拆解某一段落
- 对比不同章节的观点

### 使用示例
```
> 用户：帮我拆解《原子习惯》的第3章
> AI：好的，这就为你拆解第3章...

> 用户：我觉得第5章这个观点很棒，帮我详细解读一下
> AI：没问题，我们来深入解读这个观点...
```
