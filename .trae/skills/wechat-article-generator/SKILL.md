# 微信公众号热门文章生成器

## 功能介绍

这是一个专业的微信公众号文章生成助手，特别适合干货/教程类文章创作。

### 核心功能

1. **选题助手** - 根据领域/关键词生成热门选题
2. **大纲生成** - 自动创建文章结构框架
3. **正文写作** - 基于大纲生成完整文章
4. **模板匹配** - 自动选择适合的文章模板
5. **去AI味优化** - 让文章更像真人写作
6. **联网搜索** - 获取最新资料和数据
7. **配图建议** - 推荐合适的配图方向

### 增强功能

8. **文件保存** - 将生成的文章保存为markdown文件
9. **多格式导出** - 支持markdown、纯文本等格式
10. **版本管理** - 保存多个版本，支持对比和回退
11. **SEO优化建议** - 提供关键词、摘要等优化建议
12. **标题A/B测试** - 生成多个标题，分析点击率潜力
13. **字数统计** - 自动计算字数和预估阅读时长
14. **文章迭代** - 支持逐步修改和优化文章

### 特殊要求（重要！）

15. **固定开头** - 所有文章开头必须是：【大家好，我是快乐IT啊。】
16. **大量个人经验** - 文章中必须加入丰富的个人经历、感受、踩坑记录等
17. **作者人设** - "快乐IT啊"是一个有10年IT经验的程序员，喜欢分享，说话幽默风趣

## 使用方法

### 快速开始

1. 告诉AI你想写什么主题的文章
2. 选择或让AI自动匹配合适的模板
3. 按流程生成：选题 → 大纲 → 正文 → 优化
4. 完成后自动保存为markdown文件

### 文章模板

内置干货/教程类专用模板：

- **问题解决型**：固定开头 → 痛点 → 方案 → 步骤 → 总结
- **知识科普型**：固定开头 → 概念 → 原理 → 应用 → 案例
- **步骤教程型**：固定开头 → 准备 → 步骤1-5 → 注意事项 → 总结
- **经验分享型**：固定开头 → 背景 → 踩坑 → 经验 → 建议

### 去AI味优化策略

- 开头必须是【大家好，我是快乐IT啊。】
- 加入大量个人化表达和口语化词汇
- 适当使用语气词和感叹句
- 加入真实案例和个人经历（越多越好！）
- 使用设问句增加互动感
- 调整句式节奏，避免过于规整
- 加入"快乐IT啊"的个人故事和感受

### 工作流程

```
用户需求
    ↓
选题生成（可选联网搜索热点）
    ↓
大纲设计
    ↓
模板匹配
    ↓
正文撰写（必须包含固定开头）
    ↓
去AI味优化（加入大量个人经验）
    ↓
配图建议
    ↓
SEO优化建议 + 标题A/B测试
    ↓
字数统计 + 阅读时长预估
    ↓
保存为markdown文件（自动）
    ↓
最终成品
```
